🏆 Can interpret alternate paths in activity diagrams

Tools → UML → Activity Diagrams →

Alternate Paths

Branch nodes shows the start of alternate paths. Each control flow exiting a branch node has a guard condition : a boolean condition that should be true for execution to take that path. Only one of the guard condition can be true at any time.

Merge nodes show the end of alternate paths.

Both branch nodes and merge nodes are diamond shapes.
